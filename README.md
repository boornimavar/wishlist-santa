# Wishlist Santa

A platform that helps friends and family coordinate gift-giving for special events. Users can create wishlists for birthdays, holidays, or any occasion, and others can anonymously reserve items to prevent duplicate gifts.

## Features

- **User Authentication** – Registration and login system with username/email and password
- **Event Management** – Create events on a calendar and add wishes to each event
- **Anonymous Reservations** – Reserve gifts without revealing your identity to the event owner
- **Browse Users** – Discover friends' wishlists and upcoming events
- **Profile Management** – Edit your profile information and view your events

## Tech Stack

**Frontend:**
- React  
- Axios for API calls  
- Lucide React for icons  
- Tailwind CSS for styling  

**Backend:**
- Python Flask  
- SQLAlchemy ORM  
- SQLite database  
- Flask-CORS for cross-origin requests
